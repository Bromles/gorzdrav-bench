# Rust version

How to run:

- Install Rust SDK from the [official website](https://www.rust-lang.org/learn/get-started)
- `cargo build --release`
- Run benchmark:
  - Windows with PowerShell: `.\measure.ps1`
  - macOS/Linux: `./measure.sh`

How to clean after running:

- `cargo clean`
- Delete the `mockData` folder
